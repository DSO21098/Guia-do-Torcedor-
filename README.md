# Guia-do-Torcedor-Nome do projeto: Guia do Torcedor.
Integrantes: Davi, Diego, Kayo, Yago;
Front end: HTML e Css;
Back end: Python;
Banco de Dados: Orm;
Descrição:O Guia do Torcedor é um aplicativo criado para ajudar torcedores em geral, principalmente visitantes, oferecendo informações úteis de forma simples, rápida e organizada. O sistema foi pensado para facilitar a experiência do usuário antes, durante e após os jogos.

O aplicativo contará com funções básicas e importantes, como:

* Informações sobre partidas e campeonatos;
* Localização de estádios;
* Horários dos jogos;
* Notícias esportivas;
* Tabela de classificação;
* Informações úteis para torcedores visitantes, como transporte, alimentação e segurança próximos ao estádio;
* Área para acompanhar o time favorito.

A plataforma terá uma interface leve e fácil de usar, desenvolvida em HTML, CSS e Python, priorizando praticidade e bom desempenho. O objetivo do sistema é reunir tudo que o torcedor precisa em um único lugar, de maneira acessível e intuitiva.

Instruções básicas de uso

1. Faça login ou cadastro no aplicativo;
2. Escolha seu time favorito;
3. Acesse o menu principal para visualizar jogos, notícias e tabelas;
4. Consulte informações sobre estádios e locais próximos;
5. Utilize a pesquisa para encontrar partidas e equipes rapidamente;
6. Receba atualizações e informações importantes sobre os jogos.

O Guia do Torcedor busca oferecer uma experiência simples, funcional e útil para qualquer pessoa apaixonada por futebol.
